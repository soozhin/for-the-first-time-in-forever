知っておくと便利なgitコマンド

はじめに
  • step 1: clone
    • GitHubにある全てのファイルを自分のパソコンに入れる
  • step 2: add
    • 新しいファイルを作ったよって知らせる
  • step 3: commit
    • ディレクトリを更新する
  • step 4: push
    • GitHubにアプロードする
  • status
    • 今現在の上場を理解する
枝分かれ
  1. branch
    2. 新しい枝を作る
  2. checkout
    3. 他の枝に移動する
  3. merge
    4. 枝をマイン枝と合流させる
  4. branch -d
    5. 使い終わった枝を削除する
